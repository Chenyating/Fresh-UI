## 定义风格
以小清新风格为定位的ui组件库；
1. 风格：小清新

2. 颜色定义
```less
// 颜色

// 主色：绿色
@c-primary:#c4deaa;// 正常绿色：
@c-light-primary:#b5ddc7;// 轻绿色：
@c-dark-primary:#19be6b;// 深绿色：

// 辅助色：
@c-info:#88c1ea;
@c-success:#afcd50;
@c-warning:#a07522;
@c-error:#9b3844;

// 中性色：
@c-title:#2e5c77;;// 标题颜色：
@c-content:#2c3e50;// 正文颜色：
@c-disable:#c5c8ce;// 失效颜色
@c-bg:#ffffff;// 背景颜色
```

2. 字体、字号。

主标题，次标题、小标题
正文、辅助文字，失效文字，链接文字。

3. 过渡动画：淡入、淡出、缩放、展开折叠、旋转
